# 🧠 AI-First CRM for Healthcare Professionals (HCP)

An AI-powered Customer Relationship Management (CRM) system designed to streamline and enhance interactions with healthcare professionals. This platform combines traditional CRM capabilities with intelligent automation to improve productivity, decision-making, and compliance.

---

## 🚀 Features

### 🤖 AI-Powered Assistant

* Interactive chat assistant for managing workflows
* Generates intelligent follow-ups based on past interactions
* Enhances productivity through contextual suggestions

### 📋 Interaction Management

* Log, edit, and track interactions with healthcare professionals
* Maintain structured records for all communications
* Real-time interaction history tracking

### 📊 Data Processing & Insights

* Processes interaction data to extract meaningful insights
* Automates repetitive CRM tasks
* Improves decision-making with AI-driven recommendations

### ⚖️ Compliance & Validation

* Ensures interaction workflows follow structured compliance rules
* Validates entries to maintain data integrity

---

## 🏗️ Tech Stack

### Frontend

* React.js (Vite)
* Redux (State Management)
* CSS

### Backend

* Python
* REST APIs
* SQLite Database

### AI Integration

* LLM-based interaction (via Groq API)
* Natural language processing for chat and automation

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/ai-first-crm-hcp.git
cd ai-first-crm-hcp
```

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 3. Frontend Setup

```bash
npm install
npm run dev
```
